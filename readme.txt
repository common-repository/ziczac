Visualizza un pulsante di ZicZac per ogni tuo post!

== Description ==

Con questo plugin puoi aggiungere semplicemente un pulsante di ZicZac ad ogni tuo post.
Ogni pulsante visualizzer&agrave; il punteggio che ha ottenuto il tuo articolo.
Sono disponibili molte opzioni di visualizzazione che permettono una completa personalizzazione del sistema.

== Installation ==

1. Fai l'upload del file `ziczac.php` nella cartella `/wp-content/plugins/`
2. Attiva il plugin dal menu 'Plugins' in WordPress
